TITLE: BLOOD ON ICE

DESCRIPTION: A hockey game. I think. 
I'm starting this project over with
a new philosophy. The thing that caused the 
most problems before was the rounded corners.
I aim to fix that with new knowledge I've gained
over the past year. Hopefully I will be more
successsful this time.

STATUS: In development. Struggling.

Date: 12/23/2025

Creator(s): Narok

Version syntax is determined as follows: 
- Major releases add 1 to the x in x.0.0 number.
- Minor release that still affect the player 
experience in some visible way add 1 to the x in
0.x.0.
- Bugfixes and extremely minor changes add 1 to
the x in 0.0.x.

Versions:
